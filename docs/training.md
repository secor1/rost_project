# Процесс обучения

## Использованные датасеты

### Trash Detection V2 (основной)
- **Источник:** Roboflow
- **Изображений:** 6,000 (4,535 train / 901 val / 564 test)
- **Классов:** 18
- **Формат:** YOLOv8

### TACO Dataset
- **Источник:** Kaggle
- **Изображений:** 1,500
- **Классов:** 60
- **Формат:** COCO -> YOLO

## Параметры обучения

- model: yolo11n.pt
- epochs: 50
- imgsz: 640
- batch: 16
- device: Tesla T4 (GPU)
- optimizer: AdamW
- lr0: 0.001

| mAP50-95 | 33.6% |
